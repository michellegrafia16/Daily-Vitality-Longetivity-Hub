# Daily Vitality & Longevity Hub

A comprehensive wellness and longevity website featuring:
- Evidence-based nutrition, fitness, and mindfulness content
- Health tracking dashboard
- Affiliate product recommendations
- Contact form with Google Sheets integration

## Pages
- **index.html** - Home page with hero, features, products, testimonials
- **nutrition.html** - Nutrition guidance and meal plans
- **fitness.html** - Exercise routines and fitness tips
- **mindfulness.html** - Meditation and mental wellness
- **resources.html** - Tools, calculators, and guides
- **health-tracker.html** - Interactive health dashboard
- **contact.html** - Contact form
- **privacy.html** - Privacy policy
- **terms.html** - Terms of service
- **affiliate-disclaimer.html** - Affiliate disclosure

## Features
- Responsive design for all devices
- Dark mode support
- Health tracking with localStorage
- Product catalog with category filtering
- Newsletter signup
- Contact form with SheetDB integration

## Setup
```bash
# Install dependencies (if using server features)
npm install

# Run local server
node server.js

# Open in browser
http://localhost:3005
```

## Deployment to Vercel
1. Push to GitHub
2. Connect repo to Vercel
3. Deploy (static hosting works for HTML/CSS/JS)

## Note
The contact form requires a backend server for SheetDB integration.
